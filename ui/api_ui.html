class clubsandwich.ui.UIScene(views, *args, **kwargs)¶

Scene that renders a clubsandwich.ui-based interface.

The root view of a UIScene is always a FirstResponderContainerView. In simple terms, this means that buttons work.

Log the view hierarchy by pressing the backslash key at any time.

descendant_did_become_first_responder(view)¶

Called by FirstResponderContainerView when any descendant of this view becomes the first responder. This is so list implementations can scroll it into view.

descendant_did_resign_first_responder(view)¶

Called by FirstResponderContainerView when any descendant of this view unbecomes the first responder. This is so list implementations can release keyboard event handlers.

terminal_read(val)¶

Fires when an input event occurs, and either:

  • This view is the first responder
  • The first responder is a descendant, and no other descendants have already handled this event

You must return a truthy value if you handled the event so it doesn’t get handled twice.

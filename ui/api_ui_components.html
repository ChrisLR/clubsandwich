class clubsandwich.ui.CyclingButtonView(options, initial_value, callback, *args, **kwargs)¶

Button which, when activated, chooses the “next” value in the given list, calls its callback function with it (callback(new_value)), and updates its text to the new value.

All values must be strings. If this bothers you, pull requests are welcome.
